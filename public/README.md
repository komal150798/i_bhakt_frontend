# Digital Twin Image

**Please place your digital twin image file here with one of these names:**
- `digital-twin.png`
- `digital-twin.jpg`
- `digital-twin.jpeg`
- `digital-twin.webp`

The image should be the golden-yellow, ethereal human figure you provided. 

The code will automatically:
- Display the image as-is (no rendering)
- Apply smooth zoom in/out animation (scale 1.0 to 1.2)
- Add golden-yellow glow effect around the image

**How to add your image:**
1. Copy your image file to this directory (`frontend/public/`)
2. Rename it to `digital-twin.png` (or one of the formats above)
3. The image will automatically appear on the dashboard with zoom animation
